Hardware & IoT

Smart Seat Belt Safety System Framework

GR Yogananth, Indian Institute of Information Technology Sri City
Abstract This project presents a smart safety system combining hardware and IoT protocols to minimize vehicular collision injuries. The framework features automatic belt tightening and real-time collision detection, leveraging Bluetooth connectivity to transmit emergency alerts.

1. Introduction

Conventional seatbelt systems rely entirely on mechanical locking mechanisms during high-impact scenarios. This project explores the integration of microcontrollers and real-time sensor data to preemptively secure passengers before an impact occurs, significantly reducing the risk of whiplash and severe trauma.

2. Methodology & Tech Stack

The system architecture is built upon a central microcontroller interfacing with accelerometers and impact sensors. When anomalous deceleration is detected, the system triggers automated actuators to retract the seatbelt.

Hardware used: Accelerometer (ADXL345), Bluetooth Module (HC-05), Servo Motors, and a primary Microcontroller unit.

ElectroForge Hackathon

IoT-Based Street Light Energy Optimization Control

GR Yogananth (Runner-up, ElectroForge Hackathon)
Abstract Urban energy consumption is heavily impacted by inefficient street lighting. This system proposes an automated, IoT-driven approach to street light management, utilizing LDR sensors for dynamic dimming and the Blynk app for remote administrative override, reducing idle power waste by up to 40%.

1. Problem Statement

Traditional streetlights operate on fixed timers, leading to massive energy waste. A dynamic, responsive system is required for modern smart cities.

2. System Architecture

The core processing is handled by a NodeMCU ESP8266, which collects real-time luminosity data from distributed Light Dependent Resistors (LDR). The data is pushed to a cloud server via WiFi.
